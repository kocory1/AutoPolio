## 3. Output 정리

| Output | 설명 |
|--------|------|
| **포트폴리오** | Phase 1에서 생성된 웹 포트폴리오 |
| **Job Fit 점수** | 채용 공고 & User 프로필 간 매칭 점수 |
| **자기소개서 초안** | Writer가 생성한 초안 |
| **수정 방안 제시** | Inspector의 피드백 (Human-in-the-loop) |

---

## 4. 요약

| Phase | 핵심 흐름 |
|-------|-----------|
| **Phase 1** | GitHub 연결 + **이력서/포트폴리오 업로드** → 기본 제외 + 멀티모달 전처리 → RAPTOR 임베딩 → 에셋 생성 → User 프로필 DB |
| **Phase 2** | 채용 공고 URL (LLM 크롤링) → **Job Fit 점수화** → 자기소개서 문항 입력 → Writer 그래프(유사 샘플 → 에셋 조회) |
| **Phase 3** | 합격 자소서 DB 참고 → Writer 초안 → 유저 수정 → Inspector 보완 제안 → Human-in-the-loop 반복 |
