## 이슈 유형별 건수

* 'HTML 태그 포함'으로 잡힌 다수는 실제 HTML이 아니라 자소서에서 소제목을 꺾쇠괄호로 쓴 경우입니다.
  예: <사랑받는 회사>, <근성> → 제거 대상이 아님. 필요 시 꺾쇠괄호만 별도 처리.

## 빈 답변 vs 면접 문항
- 빈 답변 34개 모두 현재 규칙상 "면접 문항"으로 분류됨 (N분 자기소개, PT 발표, 토론하라 등).
- 일부는 문항 필드에 "면접" 버튼 텍스트가 붙어 있어 면접으로 잡힌 경우일 수 있음 (예: 146037 "로봇 어드바이저...").
- 에러 메시지 포함 ID: 사이트 전용 문구만 검사하여 현재 해당 0건 (data/jobkorea/error_message_ids.txt 참고).

## 에러/로딩 메시지 검사 기준
- 검사 패턴: "답변을 불러올 수 없", "로딩 중입니다", "잠시 후 다시 시도" (사이트 전용 안내만).
- 자소서에 자주 나오는 "오류가 발생했습니다/하였습니다"(에러 수습 경험 서술) 등은 제외함.

## 실제로 제거/보정이 필요한 것
1. 빈 답변 (34건) - 크롤링 누락 또는 면접 문항으로 답 없음
2. 매우 짧은 답변 (30건) - 일부는 의도적 짧은 답(예: "해당사항 없음")이므로 문맥 확인 필요
3. 문항이 답변 앞에 포함 (6건) - 자소서 형식상 "문항명 - 소제목" 형태로 시작한 경우, 필요 시 앞부분 정리

상세 목록은 터미널에서 다음 실행으로 확인:
  poetry run python scripts/check_jobkorea_answers.py
